# MysteryOS

### What is MysteryOS?
MysteryOS is a terminal-like OS that runs in the Linux terminal

## Version
**v0.1-alpha** — terminal-only prototype

## Overview
MysteryOS is designed to run entirely in the terminal for now, with a clean modular architecture.  
It supports basic commands and serves as the foundation for future versions that will eventually open in a dedicated terminal-like window.

## Features ( MysteryOS v0.1)
- Boot banner
- Interactive command prompt
- Commands:
  - `help` — list available commands
  - `clear` — clear the screen
  - `exit` — exit the OS
- Unknown command handling

# Dependencies
- Linux
- GCC or Clang
- Make
